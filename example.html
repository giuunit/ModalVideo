How to make it work ?

Here is the javascript you can add to your page:

$(document).ready(function() {
    //each video has need its own instance of modalVideoOptions  
    $('a[href]').each(function() {
        $(this).modalvideo(new ModalVideoOptions());
    });
});

You can also add parameters to your ModalVideoOptions object like events, more details in the modal-videos.js file
